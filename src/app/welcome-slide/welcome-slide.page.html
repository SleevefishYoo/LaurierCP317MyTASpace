This is a grading app.

MyTASpace is a grading application designed for Teaching Assistants/Instructors at Wildrid Laurier University. It focuses on creating a easier way to mark students on mobile devices.

At the moment, our app cannot view submissions related to a grade. But we would be thrilled to bring this feaure into our app made with ♥️ in the future.

We have dark mode.

Other than default color mode, MyTASpace also comes with a dark mode. It is a better user interface to use during night time. Go try it out in Settings!

More resources!

MyTASpace provides a collection of useful links for golden hawks! For instance, MyLearningSpace and Visual Schedule Builder. They all can be accessed easily within the app.

Help us improve.

If you encounter any problem while using MyTASpace, please send an error report to our development team using the Contact Us function in Settings. These reports help us improve and perfect our application over time. We really appreciate that!
