This is a grading app.

MyTASpace is a grading application designed for Teaching Assistants at Wildrid Laurier University. It focuses on creating a easier way to mark students on mobile devices.

We have dark mode.

Other than default color mode, MyTASpace also comes with a dark mode. It is a better user interface to use during night time.

Help us improve.

If you encounter any problem while using MyTASpace, please send an error report to our development team. These reports help us improve and perfect our application over time. We really appreciate that!

More resources!

MyTASpace provides a collection of useful links for Laurier students! For instance, MyLearningSpace and Visual Schedule Builder. They all can be accessed easily within the app.
